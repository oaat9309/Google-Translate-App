## Project Desc.

### API

- RapidAPI를 통해 Google Translate API 연결
- Axios를 이용한 API Fetch

### Application

- state를 활용한 모달 창 구현

### CSS

- w3c svg 이미지를 활용해 인터페이스 아이콘 추가

### Misc

- CRA를 활용한 프로젝트 setup

- ESLint / Prettier를 활용한 코드 관리
